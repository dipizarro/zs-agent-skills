# ZS Agent Skills

Repositorio de skills internas para apoyar el desarrollo de servicios Java / Spring Boot para ZS.

## Estado

Versión inicial: v0.1  
Estado: Provisional

Estas skills fueron creadas a partir de documentos recibidos progresivamente. Los estándares pueden actualizarse, reemplazarse o requerir confirmación formal por parte del líder de integración o arquitecto.

## Documentos fuente

- Buenas Practicas Java.docx
- Manual de estandares Java v_1.1.docx
- Onboarding Desarrollo Nuevos Servicios ZS.pdf
- Documento funcional de API EjecutaEjemplo.pdf
- Ejemplo Microservicio ZS.pdf

## Regla principal

Estas skills ayudan a mantener consistencia, pero no reemplazan la validación oficial del cliente.

Antes de tomar una decisión técnica crítica, validar con el líder de integración o arquitecto.

## Skills disponibles

- `zs-context`: contexto general del cliente y reglas de prioridad.
- `zs-java-standards`: estándares Java y buenas prácticas.
- `zs-new-service-onboarding`: reglas para iniciar nuevos servicios.
- `zs-api-functional-doc`: estructura para documentos funcionales de API.
- `zs-microservice-template`: lineamientos basados en el microservicio ejemplo.

## Uso recomendado

Usar estas skills con agentes de IA como Codex, Cursor, ChatGPT, Claude Code u otros asistentes de desarrollo.

Ejemplo:

> Antes de generar código Java para este cliente, revisa `skills/zs-context/SKILL.md` y `skills/zs-java-standards/SKILL.md`.
